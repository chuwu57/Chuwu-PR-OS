# judgment-hub 知识调用地图

## 第一优先级：专项 Skill 与实战判断

- `Skills/hot-public-opinion-analysis`：外部事件是否成势、实质、走势和业务启示。
- `Skills/media-narrative-analysis`：稿件或竞对传播在建立什么认知。
- `Skills/pr-contingency-plan`：上线前风险地图、口径和动作清单。
- `Skills/copy-lens-review`：文案公共语境风险。
- `Skills/press-release`：新闻稿价值、事实边界和发布风险。

## 第二优先级：框架、策略和 Workflow

- `Domains/PR/30-Principles/Principle_PR-Is-Business-Judgment.md`：PR 是经营判断，不是单纯传播执行。
- `Domains/PR/35-Frameworks/Framework_Chuwu-PR-Core-Models.md`：公关资产四分法、舆情等级三维度、危机情绪高压线、CRI/5R、媒介价值交换、单一事实源和分层语态。
- `Domains/PR/35-Frameworks/Framework_PR-Output-Quality-Evaluation.md`：问题识别、事实证据、公共语境、业务匹配、可执行性、表达质量和风险控制。
- `Domains/PR/35-Frameworks/Framework_Public-Opinion-Format-Response.md`：事实层、格式层、平台层、身份层和道德层。
- `Domains/PR/35-Frameworks/Framework_SCCT-Response-Matching.md`：责任归因、公众情绪和回应强度匹配。
- `Domains/PR/45-Workflows/Workflow_Public-Opinion-Response-Decision.md`：是否回应、回应等级、回应姿态、回应格式和风险禁区。
- `Domains/PR/45-Workflows/Workflow_Crisis-Contingency-Plan.md`：风险、触发条件、处置动作和口径。

## 第三优先级：理论校准

理论只用于校准判断，不直接写成答案主轴。

- 传播学：议程设置、新闻框架、组织-公众关系、风险沟通、新闻价值、平台化传播。
- 心理学：可得性启发、框架效应、确认偏误、社会认同、认知失调。
- 社会学：风险社会、污名、参照群体、场域、制度理论、社会运动。

## 使用边界

- 不因理论成立就直接建议动作，必须回到事实、业务目标和组织承受能力。
- 不因案例相似就机械套用，必须判断行业、品牌阶段、平台和公众情绪是否相同。
- 不把所有分歧升级成危机；只有影响资产、业务或组织信任时才升级。
