# 文案公共语境审查 / Copy Lens Review

## 中文

这个 Skill 用来做文案上线前的公共语境风险审查。

适用场景：

- 广告、社媒、节日营销、slogan、海报、短视频脚本、PR 话术。
- 判断文案能不能发、要不要改、是否需要重写。
- 识别性别、家庭、节日、饭圈、弱势群体、宠物食品监管、截图传播和媒体放大风险。

默认输出：

- 通过、修改后可用、风险较高建议重写。
- 创意意图。
- 核心风险。
- 外部视角模拟。
- 截图与标题测试。
- 修改建议。

与判断中枢的关系：

本 Skill 负责审文案表达；如果问题变成「为了传播效果要不要冒风险」，交给判断中枢拍板。

## English

This skill reviews brand copy before public release.

Use cases:

- Ads, social posts, festival campaigns, slogans, posters, short-video scripts, and PR talking points.
- Judge whether copy can be published, revised, or rewritten.
- Identify risks around gender, family, festivals, fandom language, vulnerable groups, pet-food regulation, screenshot spread, and media amplification.

Default output:

- Pass, usable after revision, or high risk and rewrite recommended.
- Creative intent.
- Core risks.
- External perspective simulation.
- Screenshot and headline test.
- Revision advice.

Relationship with the judgment hub:

This skill reviews the expression. If the question becomes whether to take a communication risk for attention, the judgment hub makes the final call.
