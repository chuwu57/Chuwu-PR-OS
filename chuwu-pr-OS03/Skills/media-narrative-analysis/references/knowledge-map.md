# media-narrative-analysis 知识调用地图

## 核心调用

- `Domains/PR/35-Frameworks/Framework_PR-Frame-Diagnosis.md`：判断稿件在使用什么框架、抬高谁、压低谁、默认什么标准。
- `Domains/PR/35-Frameworks/Framework_Chuwu-PR-Core-Models.md`：话语体系资产、媒体关系资产、媒介价值交换、PR叙事适配四变量。
- `Domains/PR/35-Frameworks/Framework_PR-Output-Quality-Evaluation.md`：问题识别、事实证据、业务匹配和表达质量。
- `Domains/PR/45-Workflows/Workflow_Brand-Positioning-Clarification.md`：当稿件涉及品牌定位或竞品心智位置时调用。
- `Domains/PR/45-Workflows/Workflow_Media-Interview-Preparation.md`：当稿件分析转为采访准备时调用。

## 理论校准

- 传播学：新闻价值、新闻框架、议程建构、消息源与记者关系、把关人理论、媒体例行程序。
- 社会学：场域、象征权力、合法性、制度理论。
- 心理学：框架效应、锚定效应、确认偏误。

## 案例参照

优先选择竞品传播、政策协作稿、危机报道、行业标准争夺和品牌叙事升级相关案例。

## 转入 judgment-hub

出现以下情况时，转入 `judgment-hub`：

- 需要决定是否回应、反制、发稿、做预案或投入资源。
- 稿件已经改变行业标准、竞品位置或我们的解释成本。
- 分析结论涉及业务动作，而不只是稿件理解。
