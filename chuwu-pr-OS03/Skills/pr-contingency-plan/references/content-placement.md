# 内容植入舆情预案

适用：品牌出现在别人的内容里——综艺、纪录片、直播、联名节目等。核心心法是把别人的内容场当成一个有规则的战场，先勘测，再铺口径。最硬、最难被替代的部分在前段（点位拆解＋机制摸底），不在话术。

## 分步结构

### 1. 舆情总目标
一句话锁定这次要守什么，给整份文档定向。写法：短，明确，后面所有判断都向它收敛。

### 2. 载体基本信息
平台、内容类型、排播时间、同台 / 同框的其他品牌。写法：清单式，客观，不带评价。

### 3. 流量盘子预估
往季或同类内容的播放、互动、话题量级。写法：给数字，再补一句量级判断——大盘还是小盘，直接决定投入规格。

### 4. 露出点位逐拆
品牌每一处露出拆出时长、位置、所含信息（片头条、花字、自然露出、彩蛋、产品使用各算多少秒、在什么剧情位置）。写法：这一格要硬，最好表格化。秒数和位置是后面所有判断的地基——广告是否恰当，往往就决定了观众吐槽与否。

### 5. 历史同类样本研究
同节目往季、同类内容里别的品牌植入怎么走：哪家因为植入生硬被骂穿全季、哪家把植入玩成了梗、官号下场互动数据如何。写法：分「弹幕 / 实时评论」和「评论区 / 跟帖」两层写，每层落一句结论——什么样的植入会被吐槽，什么样的会被接住。

### 6. 平台机制摸底
内容所在平台的实时评论 / 弹幕规则：留存逻辑、降级与精选规则、前排怎么形成、举报与屏蔽的口径。写法：把规则写成可执行的事实，能验证的去验证。这一步决定你对「环境本身」有多少把握。

### 7. 风险分维度预测
拿通用风险维度扫一遍，外加「内容向」（剧情走向、嘉宾言行、剪辑可能带来的连带风险）。写法：每个维度一句「可能被说什么」配一句应对方向。

### 8. 对手攻守推演
站到同台对手的立场，列他大概率主打的点（性价比、史低、适口 / 体验、情感故事等），再想我方怎么接住或岔开。写法：先列对方可能的牌，再写我方应对，攻守成对。

### 9. 正面引导方向（按场景）
按内容的关键场景，定每个场景希望观众关注的方向（产品利益点、公益、玩梗、好感）。写法：给方向，不给伪装成观众的成稿话术——区分「希望氛围往哪走」和「替观众写台词」，本 skill 只做前者。具体由业务自行决定执行方式。

### 10. 官方回应与客服口径库
按风险维度备好品牌方自己能公开说的话：官号互动的方向、客服统一口径。走「客观澄清 + 真实信息引导」路线。写法：一问一口径，口径要对得上事实，客观、克制。

### 11. 值班与收口
上线期谁盯什么、什么阈值上升级、官号下场节奏。写法：落成排班和触发条件，让判断变动作。

## 重心提醒
内容植入的价值集中在第 4、6 步——点位拆秒数、机制做测试。这两格做扎实，预案就立住了一大半；这两格虚，后面话术再多也是空中楼阁。
