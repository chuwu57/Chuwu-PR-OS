# 公关预案 / PR Contingency Plan

## 中文

这个 Skill 用来为一次即将上线的动作做舆情预案。

适用场景：

- 产品上新、内容植入、代言合作、规则变更、服务调整、渠道变化。
- 上线前判断会被怎么挑。
- 准备可公开署名、对得上事实的官方口径。

默认输出：

- 整体判断。
- 风险地图。
- 外部攻击推演。
- 应对口径。
- 上线前清单。

与判断中枢的关系：

本 Skill 负责把风险拆清楚；如果要在上线、延期、换表达、牺牲传播效果、保护信任之间取舍，交给判断中枢拍板。

## English

This skill prepares a public-opinion contingency plan before an action goes live.

Use cases:

- Product launches, content placements, spokesperson/KOL cooperation, rule changes, service changes, and channel changes.
- Predict what external audiences may challenge.
- Prepare official statements that can be publicly attributed and factually defended.

Default output:

- Overall judgment.
- Risk map.
- External attack simulation.
- Response lines.
- Pre-launch checklist.

Relationship with the judgment hub:

This skill maps the risks. The judgment hub makes final tradeoffs among launch, delay, message changes, communication effect, and trust protection.
