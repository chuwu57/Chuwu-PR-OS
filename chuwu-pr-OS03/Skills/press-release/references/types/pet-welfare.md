类型卡：pet-welfare

本文件仅在稿件主类型为"宠物公益类"时读取。读取后必须同时执行 `writing-rules.md`。本文件只规定类型识别、小类结构、完整稿准入和类型失败条件，不重复通用写作规则。

**宠物公益类**

触发条件

主线是"针对什么宠物相关公益议题或社会问题，采取了什么行动，形成了什么机制或结果"。

适用于流浪动物救助、动物福利倡导、养宠科普教育、行业标准建立、宠物食品安全治理、绝育/疫苗公益、领养行动等。

与相邻类型的边界

主线是宠物产品、工具、服务或权益上线，归新品发布类。
主线是数据呈现宠物消费变化，归全年回顾类。
主线依附活动节点且重点是活动结果，归活动举办类。
主线是专家或机构判断，归专家观点类。
本类可以出现数据、功能和活动，但必须服务公益对象、公益动作和机制结果。

细分小类路由

必须从以下三个小类中选择一个，不得生造新小类。

公益项目：救助计划、公益基金、领养活动、TNR（诱捕-绝育-放归）项目、义诊服务、公益捐赠。
行业标准与科普：标准发布、食安机制、养宠科普、兽医知识普及、负责任的养宠倡导。
救助行动：灾害救助、个案救助、应急响应、流浪动物收容转运。

小类规则：公益项目

适用：救助计划、公益基金、领养活动、TNR项目、义诊服务、公益捐赠等。

标题主打：公益对象 + 公益动作/机制 + 覆盖范围/成果。

标题示例：
宠物品牌启动流浪动物救助基金，首年投入500万元
平台上线领养专区，联合救助机构帮助流浪猫找到新家
宠物医院开展社区义诊，为流浪动物提供免费疫苗接种

写作结构公式：
公益对象/问题场景 → 公益行动 → 机制怎么运行 → 谁受益 → 覆盖范围或后续节点。导语第一句写针对什么公益对象或问题场景，启动、建立、捐赠或完善了什么机制；正文先克制交代公益对象，再写救助、捐赠、领养、义诊、TNR等具体动作，随后说明触发条件、执行流程或相关对象感知；结尾写后续节点或项目边界。

小类失败条件：
只有公益口号，没有具体动作或机制细节；
只写"开展公益"，不写怎么运行、谁受益；
没有说明覆盖范围或实际成果。

小类规则：行业标准与科普

适用：宠物行业标准发布、食安机制、养宠科普、兽医知识普及、负责任养宠倡导等。

标题主打：标准/科普主题 + 机制升级/发布动作 + 影响对象。

标题示例：
行业协会发布宠物食品原料标准，明确鲜肉、肉粉分级定义
宠物平台上线养宠科普专区，覆盖疫苗、绝育、行为训练等主题
品牌公开产品检测报告机制，每批次宠物粮可扫码溯源

写作结构公式：
标准/科普主题 → 发布或升级动作 → 机制怎么运行 → 谁受影响 → 覆盖范围或后续节点。导语第一句写针对什么标准或科普主题，发布、升级或建立了什么机制；正文先写标准内容或科普知识点，再写执行方式、查询路径、验证方法或传播渠道，随后说明对养宠人、品牌、经销体系的影响；结尾写后续节点或机制边界。

小类失败条件：
只有标准名称或科普口号，没有具体内容或运行机制；
只写"推动行业进步"，不写标准条款或科普知识点；
没有说明影响对象。

小类规则：救助行动

适用：灾害救助、个案救助、应急响应、流浪动物收容转运等。

标题主打：已确认事实 + 救助动作 + 边界/后续机制。

标题示例：
多地救助机构响应流浪动物过冬需求，发放保暖物资超万份
针对遗弃宠物增多现象，社区试点流浪猫TNR管理计划
宠物医院参与地震灾区动物救援，为受伤宠物提供免费治疗

写作结构公式：
已确认事实/需求 → 救助动作 → 机制解释/执行方式 → 影响范围和边界 → 后续安排。导语第一句写已公开、可确认的事件事实或需求，以及已经采取的救助或治理动作；正文只写素材中已有事实，再写救援、收容、医疗、转运、物资发放等动作，随后解释执行流程和个案边界；结尾写后续安排或继续跟进节点。

小类失败条件：
引入未经确认事实或扩大争议；
使用情绪化、对立化或定性过重表达；
只回应个案，不写机制改进或后续安排。

完整稿准入

完整稿必须同时具备：
明确公益对象；
明确公益动作；
至少一个机制细节或执行流程；
明确受益对象；
有执行流程、覆盖范围、试点范围、阶段结果或后续节点之一。

类型失败条件

命中任一项，必须先修改、降级或追问：
正文未按本小类写作结构公式的节点顺序推进，视为结构失败，必须重新组织正文后再交付。
使用"被迫、翻车、打脸、整治风波、重拳出击、强势整治"等情绪化表达；
只有公益动作，没有成效、规则、流程、机制或长期安排；
不承认公益对象或问题，直接写平台成绩；
把问题背景/痛点描述单独列小标题；
不得体现平台被动应对监管，导语及正文中禁止出现"新规发布后平台回应/响应/升级"等被动句式，应直接陈述主动公益动作和成果。
