# press-release 知识调用地图

## 核心调用

- `Domains/PR/35-Frameworks/Framework_PR-Output-Quality-Evaluation.md`：问题识别、事实证据、业务匹配、表达质量和风险控制。
- `Domains/PR/35-Frameworks/Framework_Chuwu-PR-Core-Models.md`：公关资产四分法、PR/MKT价值差异、PR商业贡献指标、技术产品传播转译。
- `Domains/PR/15-Theory/Communication/02_Theories/新闻价值.md`：判断素材是否有媒体可发性。
- `Domains/PR/15-Theory/Communication/02_Theories/消息源与记者关系.md`：判断信源、引用和媒体可用性。
- `Domains/PR/45-Workflows/Workflow_New-Product-Communication-Plan.md`：新品传播主线不清时调用。

## 理论校准

- 传播学：新闻价值、媒体例行程序、把关人理论、战略传播。
- 心理学：框架效应、权威效应、社会认同。

## 案例参照

优先选择同类通稿、发布会、公益活动、专家观点、新品发布和年度回顾案例。案例用于判断新闻角度和事实密度，不用于编造素材。

## 转入 judgment-hub

出现以下情况时，转入 `judgment-hub`：

- 新闻稿是否应发本身需要取舍。
- 发稿会放大争议、透支媒体信用或提前暴露业务动作。
- 素材能写成稿，但新闻价值不足，需要判断是否换成其他传播动作。
