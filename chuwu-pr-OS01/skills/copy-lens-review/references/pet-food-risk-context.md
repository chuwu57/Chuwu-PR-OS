# 宠物食品（猫狗）宣传雷区参考

> 供 copy-lens-review 在审核猫粮、狗粮、宠物零食、营养补充剂及相关品牌传播文案时调用。
> 依据：《广告法》第二十一条、《饲料和饲料添加剂管理条例》、农业农村部《宠物饲料标签规定》及近年行业舆情案例整理。
> 最后更新：2025年5月

---

## 一、法规红线（硬性禁止）

### 《广告法》第二十一条（饲料、饲料添加剂广告）

不得含有：

1. **表示功效、安全性的断言或者保证**（如「100%安全」「绝对健康」「放心无忧」）
2. **说明有效率或治愈率**
3. **利用科研单位、学术机构、技术专家、用户等名义作推荐、证明**（未经合规授权的专家背书、伪科普）
4. **违反安全使用规程的文字、语言或画面**
5. **法律、行政法规规定禁止的其他内容**

### 《饲料和饲料添加剂管理条例》

- **禁止**对饲料、饲料添加剂作具有**预防或者治疗动物疾病**作用的说明或宣传
- 宠物饲料广告**不得**使用「健康产品」「有利于提高健康水平」等笼统健康断言

### 《宠物饲料标签规定》第二十条（声称规则摘要）

- **禁止**对宠物饲料作具有预防或治疗宠物疾病的说明或宣传
- **所有声称必须具备证明材料**（公开发表的出版物、检测数据、试验报告等）
- 成分声称有**定量门槛**（如「××配方」≥26%、「含××配方」≥14%、「含××」≥4%），广告不得突破标签可声称范围
- 「天然」「鲜」「新鲜」等有严格原料与工艺定义，不可随意使用
- 「低脂肪」「低能量」等有水分、脂肪、能量阈值，不可口头简化
- 「处方粮」须标示「请在执业兽医指导下使用」，广告不得弱化医疗属性或暗示可自行替代诊疗
- 比较性声称（高/低/增/降）须以本企业产品为参照且变化幅度≥15%

**审核原则**：广告文案声称 ≤ 标签可声称 ≤ 有证明材料。口头承诺超出标签或证据范围，即合规风险。

## 二、功效与疗效雷区

### 绝对禁止（疾病相关）

凡暗示或明示以下效果，均属高危：

| 类型 | 典型违规表述 |
|------|----------------|
| 治疗 | 治愈、根治、消炎、抗病毒、祛泪痕药到病除 |
| 预防疾病 | 预防肾病、预防心脏病、预防猫传腹、预防胰腺炎 |
| 替代医疗 | 不用看兽医、告别处方粮、替代药物、比药有效 |
| 疗效承诺 | 三天见效、一周美毛、100%有效、包治软便 |

### 功能声称边界（可谨慎使用，须与标签一致）

标签允许对**非疾病性问题**做功能声称（如毛球、牙垢积聚），可使用「预防」字样，但须对应标签与证据。

广告中常见越界表述：

- 「美毛亮毛」「缓解泪痕」「改善肠道」「增强免疫」「 detox 排毒」——若无标签功能声称与检测支撑，易被判为功效断言
- 「处方级」「医用级」「临床级」——非处方粮不得使用医疗级暗示
- 「人食级」「婴儿级」——无公认标准，属夸大与误导

### 审核触发词

`治愈` `根治` `药效` `处方` `医用` `临床验证` `100%` `绝对` `包治` `消炎` `抗病毒` `预防疾病` `替代兽医` `人食级` `婴儿级` `排毒` `增强免疫力`

## 三、成分与溯源宣传雷区

### 核心舆论风险

近年「毒粮」「黑心粮」舆情高发，公众对以下表述极度敏感：

**鲜肉 / 肉含量**

- 宣传「鲜肉」「第一肉源」但配料实为肉粉、骨粉、水解物
- 「含肉量 80%」等数字无计算依据或与标签水分还原规则不符
- 用「鸡肉味」「牛肉味」诱食剂暗示高肉含量

**透明与溯源**

- 「进口原料」「新西兰牛肉」但无法对应标签原料组成
- 模糊「天然」「无添加」——未逐项对应标签「无××」声称
- 隐瞒氧化油脂、防腐剂、诱食剂等添加剂

**产地与代工**

- 宣传产地与实际生产地不符（代工贴牌常见舆情点）
- 只展示品牌不展示生产企业，削弱问责感

### 审核触发词

`鲜肉` `第一肉源` `含肉量` `0添加` `0防腐` `纯天然` `进口原料` `人食标准` `同源` `透明溯源` `可溯源`

### 多视角压力测试

**成分党铲屎官**：配料表、检测报告、肉源定义能否经得起逐条核对？

**打假 / 测评博主**：最夸张的一句承诺 + 配料表截图，能否构成「虚假宣传」叙事？

**竞品 PR**：能否用「你家鲜肉，我家肉粉」做对比攻击？

## 四、品类与适用边界

### 全价粮 vs 零食 vs 营养补充剂

| 品类 | 常见雷区 |
|------|----------|
| 全价主粮 | 暗示可无限量喂食、适用于所有品种/阶段而未标示 |
| 零食 | 宣传成「营养主餐」「代餐」 |
| 营养补充剂 | 宣传成「治病神药」或替代全价粮 |
| 鲜食 / 自制粮 | 「0添加」「新鲜现做」但无保质期、微生物、钙磷比说明 |

### 猫狗与生命阶段

- **猫狗混喂暗示**：猫粮给狗吃、狗粮给猫吃，或「全宠物适用」
- **阶段错配**：幼宠粮宣传给老年犬、妊娠粮泛化给所有成犬
- **品种体型**：大型犬 / 小型犬 / 品种专用须有依据，不可仅营销概念
- **绝育 / 体重管理**：涉及生理状态须有配方与标签支撑

### 审核触发词

`全阶段` `全品种` `猫狗通用` `代餐` `当主食` `无限量` `自制鲜食` `0防腐`

## 五、比较、贬低与对立雷区

### 合规与舆情双风险

- **贬低竞品**：「毒粮」「垃圾粮」「某品牌会吃死猫」——诽谤与不正当竞争
- **制造焦虑**：「不喂这个就是在害它」「穷人不配养好猫」
- **铲屎官内卷**：用消费能力、品种鄙视链、喂养方式（生骨肉 / 干粮 / 进口粮）制造对立
- **国产 vs 进口**：民族情绪与品质议题叠加，易引发站队骂战
- **无依据排名**：「全网第一」「销量冠军」「兽医首选」

### 审核触发词

`毒粮` `害人` `坑猫` `坑狗` `不配` `穷人` `智商税` `碾压` `吊打` `全网第一` `兽医推荐`

## 六、情感与人宠关系营销雷区

### 高敏感表达

宠物主对「情感绑架」「道德审判」极度反感：

- **愧疚营销**：「你没给它吃好的，对不起它」「真正的爱猫人不会选便宜粮」
- **拟人过度**：把宠物当「孩子」的同时又用「不孝」「虐待」等人类伦理词攻击用户
- **死亡恐吓**：「吃不好会短命」「你是在慢性谋杀」
- **弃养暗示**：虽非直接鼓励弃养，但暗示「养不起别养」易引动保与舆论反弹
- **丧宠消费**：在营销中过度消费宠物离世情感（节日大促尤甚）

### 家庭与性别

- 将「谁更爱猫 / 谁更懂狗」卷入夫妻、婆媳、性别对立
- 用「女朋友不如猫重要」等梗可能好笑，但截图传播易偏题

### 审核触发词

`对不起它` `不配当主人` `不爱就别养` `慢性谋杀` `害死` `虐猫` `虐狗` `道德绑架`

---

## 七、动物呈现与福利雷区

### 画面与文案

- **虐宠视觉**：强迫进食、惊吓反应、过度肥胖作为「可爱」
- **品种歧视**：贬低田园猫、串串狗、特定品相
- **活体营销**：未经同意的宠物出镜、幼宠过早离乳出镜
- **实验暗示**：未说明的「实测」「试吃实验」易引发动物福利质疑
- **野生动物元素**：错误投喂流浪动物、鼓励投喂不明食物

### 公益与救助

- 「每卖一袋捐一元」类承诺须可核实，否则涉虚假公益
- 消费救助故事须当事人同意，避免「卖惨」与摆拍质疑

## 八、渠道与表达形式特有风险

### 直播 / 短视频

- 现场承诺「今天下单送兽医咨询」——涉医疗广告边界
- 口播功效超出挂车商品标签
- 「用户反馈治好了软便」类 UGC 证言视为功效宣传

### 小红书 / 种草

- 「亲测有效」「我家猫吃了泪痕没了」——个人体验易被认定为功效暗示
- 软广不标示广告属性（另涉广告标识法规）

### KOL / 兽医合作

- 执业兽医出镜推荐特定商品须符合医疗广告与执业规范
- 「宠物营养师」「宠物博士」等头衔须可核验

## 九、高频翻车叙事模式

### 模式一：功效越界

用「美毛、去泪痕、强免疫、护肠胃」等组合暗示疗效，遇宠物发病案例时会被追溯为「误导延误治疗」。

### 模式二：成分名不副实

「鲜肉」「高肉」「0谷物」与配料表反差，是「毒粮」舆情的核心燃料。

### 模式三：焦虑 + 道德审判

在涨价、大促、新品发布时煽动「不爱宠物」愧疚，评论区易反转成「品牌吃相难看」。

### 模式四：处方粮泛化

把普通粮包装成「处方效果」，或把处方粮宣传成无需兽医指导可自行购买。

### 模式五：鲜食浪漫化

直播「鲜食」「自制」强调天然，却回避微生物、营养配比、保质期，出事后连带品牌信任崩塌。

### 模式六：极端个案代言

「流浪猫吃XX逆袭成网红猫」——个例 + 功效暗示，易被拆解为营销话术。

## 十、审核操作指引

**第一步：品类识别**
判断是全价粮、零食、补充剂、鲜食还是品牌情感传播，适用边界不同。

**第二步：法规扫描**
对照第一节、第二节，标记功效、疗效、绝对化、专家背书类表述。

**第三步：成分与证据核对**
广告声称是否能回落到标签原料组成、分析保证值及证明材料。

**第四步：情感与对立测试**
检查愧疚营销、铲屎官对立、品种歧视、死亡恐吓。

**第五步：截图传播模拟**
提取最夸张的一句 + 配料表 / 病宠图片，想象评论区叙事。

**第六步：多视角复核**
铲屎官、兽医、动保观察者、打假博主、竞品 PR 至少五视角（见 `perspective-library.md` 宠物食品扩展）。

## 十一、与通用审核的衔接

宠物食品文案除本节外，仍须走完整 copy-lens-review 流程：语义拆解、多用户视角、截图测试、媒体标题、Red Team、评分与三级结论。

**Hard Stop 叠加（宠物食品语境）**：

- 出现疾病预防 / 治疗暗示 → 不得「通过」
- 成分声称明显超出标签可声称范围 → 不得「通过」
- 使用「毒粮」等贬低竞品或恐吓用户 → 不得「通过」
- 核心叙事依赖虚假功效或虚假成分 → **风险较高，建议重写**
