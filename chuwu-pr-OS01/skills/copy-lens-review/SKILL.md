# 文案公共语境审查

## 定位

这不是创意审美评审，而是公共语境风险审查。

一条文案在内部看起来有趣，不代表进入评论区、截图、媒体标题和对手视角后仍然成立。

核心公式：

```text
公共风险 = 语义歧义 × 群体敏感度 × 场景严肃度 × 截图传播性 × 解释成本
```

只要文案靠消耗信任、制造冒犯、借弱势群体、蹭公共价值或让 PR 后续擦屁股换取短期注意力，就不是合格表达。

## 审查顺序

### 1. 先确认上下文

看清：

- 品牌、行业、渠道、发布场景。
- 目标用户和非目标用户。
- 文案目标：卖点、节日、态度、互动、转化、危机回应、品牌主张。
- 是否涉及宠物食品、猫狗产品、营养品、零食、功效暗示；涉及则读取 `references/pet-food-risk-context.md`。

### 2. 判断创意意图

用一句话写清它原本想达成什么。

然后立刻分开四层：

- 创作者想表达什么。
- 普通用户不解释能理解成什么。
- 敏感群体或非目标用户会反感什么。
- 截图和媒体标题会放大什么。

### 3. 查高风险信号

重点查：

- 性别、家庭、婚恋、生育、母职、父职。
- 未成年人、老人、弱势群体、地域、职业、阶层。
- 饭圈、黑话、暧昧梗、身体梗、疾病梗。
- 节日、灾害、公共事件、社会价值议题。
- 宠物食品里的疾病治疗、功效暗示、恐吓式养宠、配方夸大。
- 任何必须靠解释才能不冒犯的表达。

### 4. 模拟外部视角

默认读取 `references/perspective-library.md`，选择最可能误读或攻击的群体。

至少模拟：

- 普通用户。
- 非目标用户。
- 情绪最敏感的人群。
- 媒体标题。
- 竞品或恶意账号。

### 5. 判断问题层级

分清问题在哪里：

- **措辞问题**：局部词句会误读，改掉可用。
- **框架问题**：文案站位、对象关系、价值表达不稳，需要重写结构。
- **战略意图问题**：想靠冒犯、优越感、恐惧、擦边、公共价值借用获得传播，不能靠改几个词解决。

## 硬停止

出现以下情况，结论不能写「通过」：

- 有明确群体对立或冒犯风险。
- 全文自嗨、自夸，只有品牌成绩，没有用户价值。
- 核心问题无法靠局部修改解决。
- 传播效果依赖歧义、性别/家庭张力、弱势群体、疾病、阶层、地域、未成年人或灾难语境。
- 会增加品牌解释成本，消耗信任或公共合法性。
- 用短期销售、增长、互动量来证明一个公共表达值得冒险。

宠物食品文案额外硬停止：

- 暗示预防、治疗疾病。
- 配方、成分、功效超过标签或证据支持。
- 用恐惧、羞耻、内疚逼迫养宠人购买。

## 输出结构

```text
结论
通过 / 修改后可用 / 风险较高，建议重写。先给判断和一句理由。

一、创意意图
写它想表达什么，不替它美化。

二、核心风险
写 3-5 个关键风险，说明为什么会被误读或放大。

三、外部视角模拟
写普通用户、敏感人群、媒体、竞品可能怎么理解。

四、截图与标题测试
写最容易被截出来传播的一句话，以及可能被加工成什么标题。

五、公关资产判断
判断它是在增加信任、降低解释成本，还是消耗品牌信用。

六、修改建议
如果能改，给具体改法；如果方向错，直接建议重写。
```

## 三级结论

**通过**

表达清楚，符合场景情绪，无明显公共风险。最多做轻微措辞优化。

**修改后可用**

方向基本成立，但有局部误读、轻度自夸、标题不稳、价值表达不足等问题。必须写清楚改哪里、为什么改、往什么方向改。

**风险较高，建议重写**

核心站位不稳，群体风险明显，或文案本质上靠危险钩子获得传播。不要假装局部改词能解决。

## 改写原则

- 保留合理的策略意图，不保留高解释成本表达。
- 把玩笑、反差和冲突从敏感身份、家庭伦理、疾病、阶层、地域、未成年人身上移开。
- 严肃节日和公共议题提高尊重和清晰度。
- 从品牌自我表达改成用户价值表达。
- 优先写确定、真实、可被普通人理解的话。

快速结构化初筛可用：

```bash
python scripts/copy_lens_review.py input.json --format markdown
```

脚本只是第一遍，不替代最终判断。打分时读取 `references/risk-model.md`；需要例子时读取 `references/example-oppo-mothers-day.md`。

如果最后变成「为了传播效果要不要冒这个风险」，转入 `judgment-hub` 拍板。
