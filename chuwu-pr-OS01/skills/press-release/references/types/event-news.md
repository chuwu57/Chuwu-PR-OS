类型卡：event-news

本文件仅在稿件主类型为"活动举办类"时读取。读取后必须同时执行 `references/writing-rules.md`。本文件只规定类型识别、小类结构、完整稿准入和类型失败条件，不重复通用写作规则。

**活动举办类**

触发条件

主线依附于活动现场、发布会、榜单发布、颁奖、启动、表彰、签约、论坛、宠物展、开放日、专项行动落地等活动节点。

只有活动中发布、展示、形成、启动、签约、表彰或落地了新闻结果，才归为本类。活动只是承载场景时，应按真实新闻主线归入其他类型。

核心原则

借活动壳发信息，不写活动流程。活动不是新闻点；活动中发布、形成或落地了什么，才是新闻点。

与相邻类型的边界

主线是宠物产品、服务、权益或合作上线，归新品发布类。
主线是宠物消费数据变化，归全年回顾类。
主线是公益机制或救助动作，归宠物公益类。
主线是专家或机构判断，归专家观点类。
去掉活动名称后仍能成立的新闻点，应优先按真实主线归类。

细分小类路由
必须从以下三个小类中选择一个，不得生造新小类。

行业大会/展会：宠物展、大会、论坛、发布会、开放日、行业计划发布、合作发布、投入计划发布。
榜单发榜：宠物店榜单、品牌榜单、城市榜单、指南、名单、评选、颁奖盛典、年度榜单。
公益活动：领养日、义诊、流浪动物关怀、养宠科普活动、宠物亲子活动。

小类规则：行业大会/展会

适用：宠物展、大会、论坛、发布会、开放日、行业计划发布、合作发布、投入计划发布。

标题主打：活动发布结果/投入计划 + 行业对象。

标题示例：
宠物品牌宣布未来三年投入2亿元，用于流浪动物救助基地建设
宠物食品企业发布原料溯源计划，开放供应链给行业参观
宠物行业峰会发布年度趋势报告，超500家品牌参与

写作结构公式：
活动结果 → 发布内容/投入计划 → 行业对象影响 → 落地路径 → 下一步安排。导语第一句写活动中发布、宣布、启动、签约或披露了什么结果，不从嘉宾、议程或现场气氛起笔；正文先写核心结果，再写金额、周期、对象、范围、合作方、计划内容或服务方向，随后说明影响哪些行业对象及如何发生；结尾写落地路径、时间节点或覆盖范围。

小类失败条件：
标题或导语只写大会/展会举办，不写发布结果；
正文按议程、嘉宾、致辞顺序推进；
发布内容、投入计划或合作结果不清。

小类规则：榜单发榜

适用：宠物店榜单、品牌榜单、城市榜单、指南、名单、评选、颁奖盛典、年度榜单。

标题主打：榜单/指南名称 + 覆盖范围/入选数量。

标题示例：
2025年度宠物友好商家榜单发布，50城300家门店入选
宠物食品品牌口碑榜出炉，国产粮品牌占比提升至六成
城市宠物友好指数指南发布，北京、上海、成都位列前三

写作结构公式：
榜单发布结果 → 覆盖范围/入选数量 → 榜单变化或亮点 → 评选机制/入选特征 → 行业或消费参考价值。导语第一句写榜单或指南发布结果、入选数量、覆盖城市/品类/门店范围；正文先写榜单基础结果，再写今年变化或亮点，随后写评选机制、入选特征、用户评价或消费参考价值；结尾写榜单应用、地方供给变化或机制延伸。

小类失败条件：
只写颁奖盛典，不写榜单范围、数量或标准；
罗列名单过多，缺少新闻发现；
把榜单写成活动花絮。

小类规则：公益活动

适用：领养日、义诊、流浪动物关怀、养宠科普活动、宠物亲子活动。

标题主打：服务/关怀动作 + 受益对象 + 覆盖范围/结果。

标题示例：
宠物品牌开展社区领养日，帮助50只流浪动物找到新家
宠物医院联合社区开展义诊，为200只宠物提供免费体检
宠物主题公园推出亲子互动活动，普及科学养宠知识

写作结构公式：
具体支持动作 → 覆盖对象 → 申请/参与/使用机制 → 受益场景或案例 → 后续安排。导语第一句写活动提供了什么关怀、服务、科普或福利，覆盖哪些对象；正文先写具体支持动作，再写覆盖对象、参与范围、申请方式、服务内容或使用机制，随后写养宠人、宠物或社区场景；结尾写后续覆盖范围、计划或服务节点。

小类失败条件：
只写温情，不写具体保障、服务或成果；
没有受益对象、覆盖范围或实际支持内容；
现场故事压过服务结果。

完整稿准入

完整稿必须同时具备：
活动时间或地点；
活动主体；
活动中发布、启动、签约、表彰或形成的结果；
涉及对象；
数据、名单、计划、合作、机制或下一步安排之一。

类型失败条件

命中任一项，必须先修改、降级或追问：
正文未按本小类写作结构公式的节点顺序推进，视为结构失败，必须重新组织正文后再交付。
写成会务纪要；
按活动流程写；
活动本身成为主角，发布结果被弱化。
