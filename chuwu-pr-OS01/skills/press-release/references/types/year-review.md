类型卡：year-review

本文件仅在稿件主类型为"全年回顾类"时读取。读取后必须同时执行 `references/writing-rules.md`。本文件只规定类型识别、小类结构、完整稿准入和类型失败条件，不重复通用写作规则。

**全年回顾类**

触发条件

主线是"全年/阶段性数据呈现了宠物行业什么消费变化"。适用于年度宠物消费数据、品类增长、养宠人群需求变化、宠物服务消费趋势、节日节点消费变化等。

只有数据、报告、榜单或可验证现象本身构成新闻点，并能说明某类养宠人群、区域、品类或服务变化时，才归为本类。

与相邻类型的边界

主线是上线宠物产品、功能、权益或合作，归新品发布类。
主线是活动中发布结果、榜单、计划或合作，归活动举办类。
主线是专家、机构或研究报告作出的判断，归专家观点类。
主线是公益行动、救助或标准建设，归宠物公益类。
本类可以出现平台服务、品牌案例和养宠人场景，但只能解释消费变化，不得成为主线。

细分小类路由

必须从以下三个小类中选择一个，不得生造新小类。
年度消费回顾：全年宠物消费数据、年度报告、消费榜单、经营数据。
品类趋势报告：平台报告、品类数据、年度观察、细分市场报告。
节点消费热点：节日、季节、社会热点、天气事件带来的宠物消费变化。

小类规则：年度消费回顾

适用：全年宠物消费数据、年度报告、消费榜单、经营数据构成新闻点。

标题主打：报告来源/统计周期 + 核心发现 + 数据亮点。

标题示例：
《2025中国宠物消费白皮书》发布：犬猫食品消费占比超六成
宠物行业年度报告：智能用品订单同比增长八成，年轻养宠人成主力
宠物平台发布年度寻医数据，宠物体检预约量环比增长

写作结构公式：
报告来源/统计周期 → 核心发现 → 数据维度一 → 数据维度二 → 趋势边界或供给解释。导语第一句写报告名称或数据来源、统计对象、统计周期和最强发现；正文先解释核心发现对应的消费、经营、人群、区域或品类变化，再展开两个相互区分的数据维度；案例、供给变化或平台机制只能支撑数据发现；结尾写发现边界或后续观察点。

小类失败条件：
没有报告、榜单或数据来源，却写"报告显示"；
只罗列数据，没有核心发现；
把平台报告发现写成全行业结论。

小类规则：品类趋势报告

适用：宠物细分品类数据、平台观察、品类增长报告构成新闻点。

标题主打：品类/场景 + 消费变化 + 核心数据。

标题示例：
冻干主粮成为宠物食品新趋势，相关搜索量同比上涨
宠物智能用品订单增长显著，自动喂食器成养宠家庭标配
宠物保险渗透率提升，90后养宠人投保意愿最强

写作结构公式：
品类/场景聚焦 → 核心消费变化 → 人群/区域分化 → 典型案例或供给响应 → 趋势边界。导语第一句写品类或场景、统计周期、消费变化和最强数据；正文先写总变化，再展开一个细分维度（人群、区域、价位带），最后用品牌供给或服务响应解释变化；结尾收在阶段性需求特点上，无长期数据不得写长期趋势。

小类失败条件：
只写品类描述，没有消费数据或用户行为变化；
把单一品类数据外推为全行业趋势；
品牌案例篇幅超过品类趋势本身。

小类规则：节点消费热点

适用：节日、季节、社会热点、天气事件触发的即时宠物消费变化。

标题主打：热点事件/场景 + 宠物消费变化 + 核心数据。

标题示例：
春节宠物寄养需求升温，"宠物年夜饭"搜索量环比上涨
高温天带动宠物降温用品热销，冰垫、凉窝订单增长
毕业季宠物领养搜索上涨，流浪动物领养平台访问量增长

写作结构公式：
热点触发因素 → 即时消费反应 → 数据变化 → 场景/品类扩散 → 边界说明。导语第一句写热点事件、天气变化、网络话题或地域场景，以及对应宠物消费变化和核心数据；正文先写热点如何转化为搜索、订单、预订、到店或服务需求，再写变化集中在哪些品类、区域、时段或人群；供给响应只能作为支撑；结尾写短期波动、季节性或区域性边界。

小类失败条件：
只蹭热点，没有宠物消费数据；
热点叙述多于消费变化；
把短期波动写成长期趋势。

完整稿准入
完整稿必须同时具备：
明确时间窗口；
明确数据来源或统计对象；
至少一个核心数据；
至少两个趋势维度，或一个趋势维度 + 一个典型案例；
数据口径不混用；
不把平台数据、个案数据外推为全国、全行业、全社会结论。

类型失败条件
命中任一项，必须先修改、降级或追问：
正文未按本小类写作结构公式的节点顺序推进，视为结构失败，必须重新组织正文后再交付。
混用搜索量、订单量、销量、预订量等不同口径；
把平台数据、个案数据外推为全国、全行业、全社会结论；
平台功能、活动权益或品牌案例篇幅超过消费变化；
少于两个趋势维度。
