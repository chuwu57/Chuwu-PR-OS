类型卡：new-product

本文件仅在稿件主类型为"新品发布类"时读取。读取后必须同时执行 `references/writing-rules.md`。本文件只规定类型识别、小类结构、完整稿准入和类型失败条件，不重复通用写作规则。

新品发布类

触发条件

主线是"发布了什么宠物产品、谁能用、怎么用、带来什么变化"。适用于新宠物食品、新用品、新服务、新权益、新合作、新工具、新系统、新范围、新补贴、新开放能力等。

必须有明确的新产品、新服务、新权益、新合作、新范围或新动作。

与相邻类型的边界

主线是数据反映宠物消费变化，归全年回顾类。
主线依附活动节点且重点是活动结果，归活动举办类。
主线是公益行动、救助或标准建设，归宠物公益类。
主线是专家、机构或研究判断，归专家观点类。
本类可以出现数据、活动和案例，但必须服务"发布动作及对象变化"。

细分小类路由

必须从以下三个小类中选择一个，不得生造新小类。
新品上市：新宠物食品、新用品、新工具、新系统、新服务上线或升级。
品牌合作：品牌入驻、战略合作、联合定制、官方渠道、首发上线、渠道合作。
渠道推广：消费券、补贴、会员权益、节日优惠、区域活动、限时福利、试吃活动。

小类规则：新品上市

适用：新宠物食品、新用品、新工具、新系统、新服务上线或升级。

标题主打：主体 + 上市/升级 + 产品名 + 使用对象变化。

标题示例：
诚实一口推出鲜肉烘焙猫粮，蛋白含量达42%
宠物智能饮水机全新升级，支持APP远程控水与滤芯提醒
品牌上线宠物处方粮系列，覆盖肠胃、泌尿等常见健康问题

写作结构公式：
使用场景 → 上市动作 → 产品怎么用 → 解决什么养宠问题 → 覆盖范围/下一步。导语第一句写上市或升级动作、使用对象（猫/狗/特定品种）和服务变化；正文先写真实使用场景（如喂食、清洁、健康管理），再写产品入口、触发方式或操作路径，然后写效率、时效、选择、成本、安全、健康便利或服务稳定性变化；结尾收在开放范围、使用边界或下一步。

小类失败条件：
只写配方或技术能力，不写谁能用、怎么用；
只写"更健康""更智能""更营养"，没有具体使用场景；
写成产品说明书或技术介绍。

小类规则：品牌合作

适用：宠物品牌入驻、战略合作、联合定制、官方渠道、首发上线、渠道合作。

标题主打：合作双方 + 合作动作 + 服务变化/覆盖范围。

标题示例：
宠物食品品牌与连锁商超达成合作，超3000家门店上架新品
宠物医院集团与互联网平台联合推出在线问诊服务
进口宠物粮品牌入驻本土电商平台，首批覆盖一线城市

写作结构公式：
合作结果 → 双方分工 → 覆盖范围/商品服务变化 → 养宠人或经销体系获得什么 → 后续合作安排。导语第一句写合作双方、合作动作、服务变化和覆盖范围；正文先写合作已经形成的结果，再写品牌与平台/渠道分别提供什么，然后写养宠人、门店或经销侧发生的具体变化；结尾写门店、品类、区域或服务延伸。

小类失败条件：
只写"达成合作"，不写合作内容；
只写双方关系，不写服务变化；
合作分工不清，只写"共同推动"。

小类规则：渠道推广

适用：消费券、补贴、会员权益、节日优惠、区域活动、限时福利、试吃活动。

标题主打：时间/场景 + 权益内容 + 使用对象。

标题示例：
宠物品牌推出新客试吃权益，养宠人可领取50g猫粮体验装
春节宠物用品消费券上线，多地门店支持满减优惠
宠物医院推出疫苗套餐补贴，新用户可享首针优惠

写作结构公式：
时间/场景 → 核心权益 → 谁能领/怎么用 → 覆盖范围和适用边界 → 活动期限或消费场景。导语第一句写活动时间、活动主体、核心权益、适用对象和使用场景；正文先写权益内容，再写领取、使用、适用城市/品类/门店/场景，随后补充对应消费需求；结尾收在活动期限、适用边界或后续安排，不得包装成长效能力。

小类失败条件：
只写补贴金额，不写对象、时间、范围和规则；
只写活动口号，不写养宠人如何获得权益；
写成促销海报。

完整稿准入

完整稿至少满足以下 6 项中的 4 项：
发布主体；产品/服务/权益/合作名称；
使用对象；使用场景或操作路径；
上市、升级、开放、试点或覆盖状态；
带来的具体变化。
缺少三项以上，不得写完整稿。

类型失败条件

命中任一项，必须先修改、降级或追问：
正文未按本小类写作结构公式的节点顺序推进，视为结构失败，必须重新组织正文后再交付。
只写配方或技术能力，不写怎么用；
只写怎么用，不说解决了什么需求；
只写合作，不写服务变化；
只写补贴，不写对象、时间、范围和规则；
写成企业介绍、产品说明书、战略宣言或促销海报。
