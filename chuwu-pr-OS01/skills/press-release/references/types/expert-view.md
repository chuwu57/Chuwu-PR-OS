类型卡：expert-view

本文件仅在稿件主类型为"专家观点类"时读取。读取后必须同时执行 `references/writing-rules.md`。本文件只规定类型识别、小类结构、完整稿准入和类型失败条件，不重复通用写作规则。

**专家观点类**

触发条件

主线由宠物专家、兽医、营养师、机构、报告、榜单、研究、媒体评论、券商观点、行业协会或公开数据中的判断驱动。

只有观点、判断、研究发现或行业解释构成主线时，才归为本类。专家只出现一句评价，但主体仍是品牌动作、新品发布或活动举办时，不归本类。

企业动作只能作为佐证，不得成为第一主语。品牌实践原则上不超过全文 40%。

与相邻类型的边界

主线是品牌发布、合作、权益或工具，归新品发布类。
主线是活动中发布结果，归活动举办类。
主线是宠物消费数据变化，归全年回顾类。
主线是公益动作、规则或机制，归宠物公益类。
本类第一主语必须是观点来源、报告来源或研究发现，不是企业动作。

细分小类路由

必须从以下两个小类中选择一个，不得生造新小类。
专家观点：宠物专家、兽医、营养师、行业人士、媒体评论、公开观点中的判断构成主线。
研究报告：宠物行业报告、调查、券商研究、机构数据、榜单、研究发现、公开数据分析。

小类规则：专家观点

适用：宠物专家、兽医、营养师、行业人士、媒体评论、公开观点中的判断构成主线。

标题主打：观点来源 + 核心判断 + 行业/对象变化。

标题示例：
宠物营养师：夏季宠物饮水量应增加三成，干粮喂养需注意补水
业内人士：国产宠物粮市场份额持续提升，品质升级成关键
兽医专家提醒：宠物定期体检可早发现80%以上慢性疾病

写作结构公式：
观点来源 → 核心判断 → 判断依据 → 影响分析 → 实践佐证 → 边界提示。导语第一句写谁提出了什么判断，涉及什么行业、群体、区域或议题；正文先写观点来源和核心判断，再写数据、案例、公开现象、市场变化、行业背景或专家原话作为依据，随后解释对行业、品牌、养宠人、宠物医院或区域市场的影响；品牌实践只能作中后段佐证；结尾写观点边界或后续观察点。

小类失败条件：
没有来源却写"专家认为"；
让专家替品牌做广告；
把个人观点写成行业共识。

小类规则：研究报告

适用：宠物行业报告、调查、券商研究、机构数据、榜单、研究发现、公开数据分析。

标题主打：报告/机构来源 + 核心发现 + 数据亮点。

标题示例：
高校研究报告：城市流浪猫密度与社区TNR覆盖率呈负相关
宠物行业白皮书：宠物医疗消费年均增长15%，体检渗透率仍低
券商研报：宠物食品赛道持续扩容，头部品牌集中度提升

写作结构公式：
报告来源 → 核心发现 → 关键数据和口径 → 细分维度 → 行业解释/实践佐证 → 边界提示。导语第一句写报告或机构来源、统计对象、统计周期和核心发现；正文先写核心发现及其对应的行业、群体、区域、场景或经营变化，再写关键数据和统计口径，随后展开一到两个细分维度；行业解释、公开案例或品牌实践只能服务报告发现；结尾写适用范围、边界或审慎判断。

小类失败条件：
没有报告或机构来源，却写"报告显示"；
没有统计对象、统计周期或数据口径；
把报告发现拔高为确定趋势。

完整稿准入

专家观点类完整稿必须同时具备：
明确观点来源；
明确来源身份或角色；
明确核心观点和数据；
有事实、数据、案例、报告或公开现象支撑；
有观点边界或适用范围。

研究报告类还必须具备：
报告或机构来源；
统计对象、周期或数据口径。

类型失败条件

命中任一项，必须先修改、降级或追问：
正文未按本小类写作结构公式的节点顺序推进，视为结构失败，必须重新组织正文后再交付。
没有来源却写"专家认为"；
把个人观点写成行业共识；
把机构判断写成政策要求；
连续机械使用"报告显示"等词语；
让专家替品牌做广告；
连续堆叠专家引语无叙述过渡；
没有明确第三方观点来源；
品牌成为第一主语；
编造专家姓名、机构、职务、身份或直接引语；
使用"权威专家力挺""专家点赞""行业一致认为"等站台式表达；
把研究报告写成品牌宣传稿。
