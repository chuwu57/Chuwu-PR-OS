# 新闻通稿 / Press Release

## 中文

这个 Skill 用来写、改和检查新闻通稿。

适用场景：

- 新品发布、宠物公益、活动举办、专家观点、全年回顾。
- 起草新闻稿、改稿、局部重写、终稿检查。
- 判断素材是否足以支撑一篇外部媒体稿。

默认输出：

- MVP 方向或完整通稿。
- 局部改写。
- 终稿风险检查。
- 是否缺事实、缺新闻点、缺外部读者价值。

与判断中枢的关系：

本 Skill 负责把事实写成稿；如果要判断值不值得发、发了是否消耗信用，交给判断中枢拍板。

## English

This skill drafts, edits, and checks press releases.

Use cases:

- New product releases, pet welfare stories, events, expert views, and year-end reviews.
- Drafting, editing, partial rewriting, and final review.
- Judging whether the material can support an external media release.

Default output:

- MVP direction or full press release.
- Partial rewrite.
- Final risk review.
- Checks for missing facts, weak news value, and lack of reader value.

Relationship with the judgment hub:

This skill turns verified facts into a release. The judgment hub decides whether the release is worth publishing and whether it may consume brand credit.
