# [热点名称] 舆情研判

## 1. 是否已经形成舆情

**结论**：[已形成舆情 / 尚未形成舆情 / 处于舆情前夜]

**判断依据**：
- 源头：[最早信息来源]
- 扩散链路：[是否多平台传播、媒体跟进、KOL二次解读]
- 讨论规模：[是否超出小范围圈层]
- 媒体类型：[当前主导传播的媒体/平台类型]
- 业务相关性：[与本业务的关联度]

> 若结论为"尚未形成舆情"，简要说明原因后可不进入后续完整研判，给出监测建议即可。

## 2. 舆情实质分析

**这个舆情为什么会出现**：[重点回答：事件冒犯了谁/谁发起了争议？背后映射的是哪些社会矛盾、群体对立或价值观冲突？为什么此时此刻能引发集体情绪？——须经过深度质疑子agent挑战后的修订结论，不能停留在表面归因]

**深层解读**：[用一两句大白话说清"穿透表象的洞察"是什么，括号内点出理论出处即可。例如："说白了，用户自己开玩笑和品牌替用户开玩笑是两码事——谁说的比说了什么更重要（传播学'编码/解码'理论）。"]

**关键事实与待确认信息**：
- 已确认：[列出]
- 待确认：[列出]

**公众情绪焦点**：[愤怒/同情/焦虑/质疑集中在哪里，以及这种情绪的社会根源]

**本次舆情值得关注的新特点**：
- [新特点1]
- [新特点2]

**对相关方的长期影响**：[品牌心智、行业认知、政策语境、监管关注等方面的中长期影响判断]

## 3. 舆情走势研判

**当前阶段**：[潜伏期 / 激化期 / 爆发期 / 反转期 / 降温期]

**判断依据**：[结合媒体性质、案例规模、标签是否成型、官方是否介入]

**升级触发器**：[哪些事件或动作可能使舆情升级]

**降温条件**：[什么情况下可能逐步降温]

**当事方可能动作**：[回应/澄清/道歉/沉默/反击/整改等]

**媒体可能报道角度**：[追问责任/复盘商业模式/类比同行/引入监管/挖旧账等]

> 不做绝对预测，使用条件化判断。必要时引用历史类似案例辅助判断。

## 快速版要求

当用户要求"快速版""一句话判断""老板能看版"时，保留四段结构，每段压缩到1-3句话，总字数控制在800字以内。不省略"待确认信息"和"待讨论的问题"。
