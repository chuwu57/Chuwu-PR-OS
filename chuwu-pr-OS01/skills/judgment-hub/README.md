# 判断中枢 / PR Judgment Hub

## 中文

这个 Skill 用来做最终判断和取舍。

适用场景：

- 是否回应、是否发稿、是否延期、是否调整动作。
- 风险和业务目标冲突。
- 声量、信任、资源、时机之间需要排序。
- 其他专项 Skill 已经完成分析，但还缺最后结论。

默认输出：

- 结论。
- 问题性质。
- 取舍逻辑。
- 推荐动作。
- 不建议做什么。

与其他 Skill 的关系：

热点、稿件、预案、新闻稿、文案审查负责提供输入；判断中枢负责把输入变成最终决策。

## English

This skill makes final PR judgments and tradeoffs.

Use cases:

- Decide whether to respond, issue a release, delay, or change an action.
- Handle conflicts between risk and business goals.
- Prioritize among attention, trust, resources, and timing.
- Convert analysis from specialized skills into a final recommendation.

Default output:

- Decision.
- Nature of the problem.
- Tradeoff logic.
- Recommended action.
- What not to do.

Relationship with other skills:

The specialized skills provide inputs. The judgment hub turns those inputs into a final decision.
