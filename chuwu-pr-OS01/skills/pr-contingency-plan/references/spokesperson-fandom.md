# 代言人 / 粉圈预案

适用：明星 / 达人代言、IP 合作。这类预案有一半篇幅在做「粉圈是什么」的测绘，因为这是品牌方最容易踩空的地方——品牌按常规营销逻辑出牌，往往不知道粉圈自有一套情绪和动员规则。

## 分步结构

### 1. 合作背景与目标
为什么合作、想要什么（曝光、形象绑定、销量）。写法：短，定方向。

### 2. 粉圈生态测绘
给不懂粉圈的人补课，但每段落点都是「所以风险在哪」。覆盖：
- **平台分布与用法**——粉丝主要在哪、各平台干什么用。
- **内部结构**——核心（粉头 / 站姐）、次核心（小组管理）、散粉，各自的话语权和行为模式。
- **动员机制**——靠什么话术 / 约束动员，短期爆发力有多强。
- **CP / 唯粉等派系**——派系矛盾是品牌物料最常踩的雷区。

写法：像科普，但每条都回答「这对品牌意味着什么风险」。

### 3. 合作对象风险档案
全维度建档：出道 / 团体史、代言史、近期公众事件、长期公众形象、粉丝画像。写法：每条都要回答「这对品牌意味着什么」——光罗列经历没用，要落到风险或机会。

### 4. 敌对方与恶意图谱
竞争格局（同代竞品、对家）、常见黑梗、敏感词、黑粉构成与行为。写法：把雷标清楚，免得物料无意识踩中。敏感词单列成表，供物料审核直接比对。

### 5. 核心关注点收敛
把一大堆可能的雷，收成几条能记住的。代言类常见的核心关注点：粉丝情感需求没被满足（背刺、不尊重）、品牌执行细节失误、代言头衔不匹配、对家粉丝干扰。再定几条宣传红线（不碰哪些词、不提哪些人 / 哪些过往）。写法：这一步最见功力——预案做厚不难，收成几条能执行的判断才难。

### 6. 全周期分阶段风险
准备 / 预热 / 官宣 / 传播，每阶段各自的雷。写法：分阶段列，每阶段三五条，配应对方向。

### 7. 次生风险
粉丝行为带来的衍生问题：私生追踪导致物料泄露、内场名额 / 特殊周边倒卖、肖像与双人元素的侵权争议等。写法：列出来，标明哪些是品牌该提前规避的。

### 8. 应对口径与执行建议
官方文案的统一口径（聚焦代言人个人、照顾主要粉群情绪、避开派系雷区）、特殊情况的处理通路（与艺人团队 / 内容 / 公关三方对口径）。写法：把前面所有判断收成几条品牌方真能做的动作。

## 重心提醒
代言类的「对象研究」要先研究一整个生态（第 2 步），这是它和别的预案最不同的地方。最聪明的一处永远是第 5 步的收敛——把雷收成几条记得住的红线，比铺一地风险点有用得多。长期还要做尾部监控：合作后持续关注代言人的个人争议和重大过失。
