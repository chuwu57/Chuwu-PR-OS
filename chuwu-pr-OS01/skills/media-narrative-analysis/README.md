# 稿件分析 / Article Narrative Analysis

## 中文

这个 Skill 用来分析一篇稿子在替谁建立什么认知。

适用场景：

- 分析媒体报道、公关稿、自媒体文章、竞对传播、政策通报。
- 判断一篇稿是否是软文、通稿、批评性报道或政策协作内容。
- 判断它对本公司是否有影响，以及是否需要动作。

默认输出：

- 传播意图。
- 核心断言。
- 叙事动作。
- 对公关资产和业务解释成本的影响。
- 不应对、观察或进入判断中枢的建议。

与判断中枢的关系：

本 Skill 负责读懂稿子；如果要决定是否回应、反制、发稿或投入资源，交给判断中枢收口。

## English

This skill analyzes what cognition an article is trying to build and for whom.

Use cases:

- Analyze media reports, PR articles, self-media posts, competitor narratives, and policy communications.
- Identify sponsored content, syndicated narratives, critical reporting, or policy-aligned corporate messaging.
- Judge whether the article affects the company and whether action is needed.

Default output:

- Communication intent.
- Core claim.
- Narrative moves.
- Impact on PR assets and explanation cost.
- Recommendation: no response, monitor, or escalate to the judgment hub.

Relationship with the judgment hub:

This skill reads the article. The judgment hub decides whether to respond, counter-message, issue a release, or allocate resources.
