# chuwu-pr-OS01

## 中文

这是初五的 PR 判断 Skill 包，包含 6 个 Skill：

- `judgment-hub`：判断中枢。
- `hot-public-opinion-analysis`：热点舆情分析。
- `media-narrative-analysis`：稿件分析。
- `pr-contingency-plan`：公关预案。
- `press-release`：新闻通稿。
- `copy-lens-review`：文案公共语境审查。

设计原则：

- PR 是业务判断，不是材料处理。
- 信任高于情绪，长期高于短期。
- 先判断问题性质，再决定是否动作。
- 输出要能帮业务减少解释成本，而不是制造更多文档。
- 专项 Skill 负责具体任务，判断中枢负责最终取舍。

文件规则：

- 顶层 README 为中英双语。
- 每个 Skill 的 README 为中英双语。
- 每个 `SKILL.md` 保持中文。
- `Changelog.md` 本次保持空白。

## English

This is Chuwu's PR judgment skill package. It includes 6 skills:

- `judgment-hub`: judgment hub.
- `hot-public-opinion-analysis`: hot public opinion analysis.
- `media-narrative-analysis`: article narrative analysis.
- `pr-contingency-plan`: PR contingency planning.
- `press-release`: press release drafting and review.
- `copy-lens-review`: public-context copy review.

Design principles:

- PR is business judgment, not material processing.
- Trust matters more than emotion; long-term value matters more than short-term noise.
- Judge the nature of the problem before deciding whether to act.
- Outputs should reduce business explanation cost, not create more documents.
- Specialized skills handle specific tasks; the judgment hub makes final tradeoffs.

File rules:

- The top-level README is bilingual.
- Each skill README is bilingual.
- Each `SKILL.md` remains in Chinese.
- `Changelog.md` is intentionally blank for this release.
