# 初五的公关判断 · chuwu-pr-judgment

> 一个让 AI 用「初五」的逻辑思考公关问题的 Claude Skill。

装上之后,AI 遇到任何品牌公关、舆情、危机、传播、竞品、媒体关系、职业发展类的问题,都会先按初五的判断框架过一遍——**能答的给专业打法,没遇过的也按这套逻辑和风格诚实推演,并标明哪里是推测。**

它不是一个查资料的工具,是一个**判断的底座**。给它一个公关难题,它替你先想一遍「初五会怎么拍板」。

## 初五是谁

一位 INTJ 消费品品牌公关,在公众号「初五的公关来信」和小红书持续写公关方法论、舆情拆解、行业观察。本 skill 从其近百篇原创内容里提炼判断逻辑,核心信念一句话:

> 先看本质,再排优先级,敢下结论,认现实的残酷,守专业的底线。
> 信任高于情绪,长期高于短期,被信任的确定性高于被喜欢。

## 安装

这是一个标准的 Claude Agent Skill,不含可执行脚本,放进 skill 目录即可生效。

**Claude Code / Claude 桌面端:** 把这个仓库整个下载下来(Code → Download ZIP,或 git clone),把文件夹**重命名为 `chuwu-pr-judgment`**,放进 skill 目录:

```
# 用户级(全局生效)
~/.claude/skills/chuwu-pr-judgment/

# 或项目级(仅当前项目)
<你的项目>/.claude/skills/chuwu-pr-judgment/
```

放好后目录里应直接有 `SKILL.md`。重启或新开会话即可生效。其他支持 Skills 的客户端,按其导入方式把这个文件夹作为一个 skill 导入;核心文件是 `SKILL.md`,开头的 YAML(name / description)是触发用的,不要删。

**验证生效:** 问它「一条小红书吐槽我们产品,要不要发官号声明?」如果它先给本质判断、再给优先级和取舍,而不是罗列一堆四平八稳的步骤,就对了。

## 适合什么场景

- **舆情/危机:** 要不要回应、怎么定舆情等级、口径怎么改、媒体来求证但内部没消息怎么接、要不要冷处理、该不该自爆瑕疵、真投诉还是黑公关、合作伙伴爆雷怎么办、媒体只盯细节负面怎么办。
- **传播策略:** 年度策略怎么写(IECN 模型)、这波传播算不算带来生意增长、素材和渠道怎么取舍、怎么把传播翻译成渠道动作。
- **竞品与行业:** 竞品在干什么、怎么从财报建立竞品认知、怎么练研究手感。
- **媒体关系:** 跨行新人怎么建媒体关系、平淡期怎么维护、媒体资产怎么管。
- **代言/粉圈:** 明星代言前公关要防什么(17 块预案清单)。
- **职业/团队:** 公关怎么定位自己、新人怎么成长、团队怎么带。
- **没遇过的新问题:** 也会按初五的逻辑给出有取舍的建议,并诚实标明哪里是推测、缺什么信息。

## 不负责什么

外部热点未来怎么走、拆解某一篇具体报道、文案上线前的合规审查、纯文字风格润色——这些是另外的工具该做的事。
